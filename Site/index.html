JSON Cocoa Framework

This JSON framework does not add any stand-alone classes. It is a collection of categories that adds methods to existing classes. Importing <JSON/JSON.h> and linking against the framework adds the following methods:

  • -(id)objectFromJSON;
-(id)objectFromJSONFragment;

    Added to NSString. These return the Objective-C data structure described by the JSON text held within the string.

  • -(NSString*)JSONString;

    Added to NSArray and NSDictionary. This returns JSON text describing the (possibly nested) data structure. In JSON the keys of a dictionary (aka "object") must be strings. In Objective-C they need not be, but this method will fail otherwise.

  • -(NSString*)JSONStringFragment;

    Added to the NSNull, NSNumber & NSString classes.

JSON has to be encapsulated by an array or dictionary, so a fragment is not valid JSON on its own. Pretending it is can be quite convenient, hence the methods to deal with fragments.
